11class StreamRequestHandler[StreamReqT: StreamRequest[Any]](StreamHandlerBaseMixin[StreamReqT], ABC): 

12 """Abstract base class for synchronous streaming request handlers. 

13 

14 The sync mirror of `pymediate.StreamRequestHandler`: a stream handler processes a 

15 `StreamRequest` by **yielding** a stream of chunks, consumed lazily via the sync 

16 `Mediator.stream()`. It only needs to specify the request type - the chunk type is 

17 inferred from the request's `StreamRequest[ChunkT]` declaration. 

18 

19 The handler performs class-definition-time validation via __init_subclass__ to ensure: 

20 - The __call__ method exists and is a generator (uses `yield`) 

21 - The __call__ parameter annotates the exact declared request type 

22 (not a base class or union) 

23 - The __call__ return type is `Iterator[ChunkT]`, matching the request's chunk type 

24 

25 This validation happens at class definition time (import time), catching errors 

26 early rather than at runtime. 

27 

28 Type Parameters: 

29 StreamReqT: The type of stream request this handler processes. Must inherit 

30 from StreamRequest; static type checkers enforce the bound, and PyMediate 

31 validates it at class definition time. 

32 

33 Examples: 

34 Streaming tokens for a completion request: 

35 ```python 

36 from collections.abc import Iterator 

37 from dataclasses import dataclass 

38 from pymediate.sync import StreamRequest, StreamRequestHandler 

39 

40 @dataclass 

41 class StreamCompletion(StreamRequest[str]): 

42 prompt: str 

43 

44 class CompletionHandler(StreamRequestHandler[StreamCompletion]): 

45 def __call__(self, request: StreamCompletion) -> Iterator[str]: 

46 yield from request.prompt.split() 

47 ``` 

48 

49 Stream handler with dependencies: 

50 ```python 

51 class ExportRowsHandler(StreamRequestHandler[ExportRows]): 

52 def __init__(self, database: Database): 

53 self.database = database 

54 

55 def __call__(self, request: ExportRows) -> Iterator[Row]: 

56 for row in self.database.iter_rows(request.table): 

57 yield row 

58 ``` 

59 

60 Note: 

61 For asynchronous stream handlers, use `pymediate.StreamRequestHandler` 

62 instead. The `__call__` must be a generator (contain `yield`); a plain `def` 

63 that returns an iterator is rejected at class definition. Pipeline behaviors do 

64 not wrap `stream()`. 

65 

66 Raises: 

67 InvalidStreamRequestTypeError: If the request type doesn't inherit from StreamRequest. 

68 InvalidHandlerSignatureError: If __call__ isn't a correctly typed generator. 

69 

70 See Also: 

71 - StreamRequest: Base streaming request class. 

72 - Mediator.stream: Routes stream requests to stream handlers (sync version). 

73 - pymediate.StreamRequestHandler: Async stream handler variant. 

74 """ 

75

80 """Handle the request and yield a stream of chunks. 

81 

82 This is an abstract method that must be implemented by all sync 

83 StreamRequestHandler subclasses as a generator, with the signature 

84 `def __call__(self, request: RequestType) -> Iterator[ChunkType]: ...` 

85 and at least one `yield`. 

86 

87 Args: 

88 request: The stream request to handle. 

89 

90 Yields: 

91 Chunks of the type declared by the request's `StreamRequest[ChunkType]`. 

92 

93 Note: 

94 The annotation must be the exact request class - a base class or union 

95 passes static checking (contravariance) but raises 

96 `InvalidHandlerSignatureError` at class definition. The method must be a 

97 generator; for async stream handlers, use 

98 `pymediate.StreamRequestHandler`. 

99 """
