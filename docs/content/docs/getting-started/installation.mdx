---
title: Installation
description: Requirements and install options for PyMediate on Python 3.12+.
---

PyMediate is published on [PyPI](https://pypi.org/project/pymediate/) and can be installed with a
Python package manager.

## Requirements

- Python **3.12 or higher** — PyMediate uses [PEP 695](https://peps.python.org/pep-0695/) type parameter syntax.
- The core package has no required dependencies.
- The optional `di` extra installs `dependency-injector>=4.41.0` for container integration.

## Install the core package

<Tabs items={['pip', 'uv', 'poetry']}>
  <Tab value="pip">
    ```bash
    pip install pymediate
    ```
  </Tab>
  <Tab value="uv">
    ```bash
    uv add pymediate
    ```
  </Tab>
  <Tab value="poetry">
    ```bash
    poetry add pymediate
    ```
  </Tab>
</Tabs>

## Install with dependency injection support

The `di` extra adds the [`dependency-injector`](https://python-dependency-injector.ets-labs.org/) integration:

<Tabs items={['pip', 'uv', 'poetry']}>
  <Tab value="pip">
    ```bash
    pip install 'pymediate[di]'
    ```
  </Tab>
  <Tab value="uv">
    ```bash
    uv add 'pymediate[di]'
    ```
  </Tab>
  <Tab value="poetry">
    ```bash
    poetry add 'pymediate[di]'
    ```
  </Tab>
</Tabs>

<Callout type="info">
  `DependencyInjectorServiceProvider` only imports successfully when `dependency-injector` is
  installed. Without the `di` extra, `from pymediate.providers import
  DependencyInjectorServiceProvider` raises `ModuleNotFoundError`. See
  [troubleshooting](/docs/guide/troubleshooting#installation-problems)
  if you hit this.
</Callout>

## Verify the install

```python
import pymediate

print(pymediate.__version__)
```
