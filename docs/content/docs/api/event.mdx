---
title: Event
description: API reference for the Event base class.
---

```python
from pymediate import Event

class Event: ...
```

`Event` is the base class for messages published with
[`Mediator.publish()`](/docs/api/mediator#publish). Publishing delivers an event to every
registered handler for its exact class and returns `None`. Zero handlers is valid.

Events have no response type parameter. The same `Event` class is re-exported from `pymediate`
and `pymediate.sync`. Its subscription registry is also shared, so all handlers for one exact
event type must be asynchronous or all must be synchronous.

## Example

```python
from dataclasses import dataclass

from pymediate import Event


@dataclass(frozen=True)
class OrderPlaced(Event):
    order_id: int
    item: str
```

Dispatch is exact: a handler registered for a base event class does not receive instances of a
derived event class.
