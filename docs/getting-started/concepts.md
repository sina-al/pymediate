# Basic Concepts

Learn the core concepts of PyMediate.

## Requests

Requests represent intentions - things you want to do.

## Responses

Responses contain the results of processing a request.

## Handlers

Handlers contain the business logic for processing requests.

## Mediator

The Mediator routes requests to their handlers.

## Resolvers

Resolvers find the right handler for each request type.
